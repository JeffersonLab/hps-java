10  /**
11   * JNA Wrapper around the GBL C++ dynamic library
12   * <p>
13   * All of the functions in this interface are bound to un-mangled wrapper functions
14   * in the GBL C++ dynamic library. This wraps GBL v3.1+ of the C++ library.
15   * <p>
16   * <code>
17   * git clone --depth 1 --branch V03-01-00 https://gitlab.desy.de/claus.kleinwort/general-broken-lines.git
18   * <p>
19   * cmake -B general-broken-lines/build -S general-broken-lines
20   * <p>
21   * cmake --build general-broken-lines/build
22   * </code>
23   * <p>
24   * Then make sure the full path of the GBL library is either in a system path (e.g. /usr/local/lib),
25   * or you tell JNA where the GBL library is with
26   * <code>
27   * -Djna.library.path=/full/path/to/general-broken-lines/build/lib
28   * </code>
29   * <p>
30   * The documentation for the GBL C++ library also has a page on how to access it
31   * <a href="https://www.desy.de/~kleinwrt/GBL/doc/cpp/html/jnausage.html">via JNA</a>.
32   */
